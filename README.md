# ゲームの説明
1.ケータイをスワイプをしてボールを下に落としてください。
2.次のステージに進むたびスコアが加算されていきます。
3.ドクロマークにボールが当たった時点でゲーム終了です。
4.ゲーム終了時のスコアが最終的なスコアとなります。
5.ハイスコアを目指し、友達と競い合ってください。
